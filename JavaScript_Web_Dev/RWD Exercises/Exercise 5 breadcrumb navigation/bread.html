ANGULAR

Angular is a framework, for creating enterprise level client side application.Client side and server side is decided based on where the code gets executed. Angular uses Typescript as it's language for implementation. It help us to create very dynamic and with best user experience web applications. It contains modules and components, where modules is a collection of components and an application can consists of many modules. Angular framework is maintained by GOOGLE. It provides more features as compared to other frameworks or libraries.
